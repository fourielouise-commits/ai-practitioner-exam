Domain 1: Fundamentals of AI and ML (20%) — 17 Questions
Q1
Which type of machine learning uses labeled input-output pairs during training?
Show Answer
B. Supervised learning. Supervised learning trains models on labeled data where both inputs and correct outputs are provided, enabling the model to learn the mapping function.
Q2
A model performs well on training data but poorly on unseen data. What is this called?
Show Answer
B. Overfitting. The model memorized training noise instead of learning generalizable patterns. Techniques like regularization, dropout, and early stopping help prevent this.
Q3
Which metric is BEST suited for evaluating a binary classification model when the dataset has severe class imbalance?
Show Answer
B. F1 Score. F1 balances precision and recall, making it more informative than accuracy when one class vastly outnumbers the other.
Q4
Which AWS service provides pre-trained APIs for common ML tasks like text analysis, translation, and image recognition without requiring ML expertise?
Show Answer
B. AWS AI Services. Services like Comprehend (NLP), Translate (language), Rekognition (vision), and Polly (speech) provide ready-to-use APIs that require no ML expertise.
Q5
What is the purpose of a validation dataset in the ML pipeline?
Show Answer
B. To tune hyperparameters and prevent overfitting during training. The validation set provides an unbiased evaluation during training to guide model selection. The test set is reserved for final evaluation.
Q6
A retail company wants to group customers by purchasing behavior without predefined categories. Which ML approach is most appropriate?
Show Answer
C. Clustering. Clustering is an unsupervised technique that groups data points by similarity without requiring predefined labels. K-means and DBSCAN are common algorithms.
Q7
Which of the following describes the concept of "inference" in machine learning?
Show Answer
B. Using a trained model to make predictions on new data. Inference is the production phase where the trained model processes new inputs and generates outputs.
Q8
What is the primary difference between AI, machine learning, and deep learning?
Show Answer
B. AI is the broadest concept; ML is a subset of AI; deep learning is a subset of ML. AI encompasses any machine mimicking intelligence. ML achieves this through learning from data. Deep learning uses neural networks with multiple layers.
Q9
Which technique helps reduce the number of input features while preserving the most important information?
Show Answer
B. Dimensionality reduction. Techniques like PCA reduce feature count while retaining the variance that matters most, improving model performance and reducing computation.
Q10
A model that is too simple to capture the underlying patterns in the data suffers from:
Show Answer
B. High bias (underfitting). An underfit model makes strong assumptions that oversimplify the data, resulting in poor performance on both training and test sets.
Q11
Which AWS service allows you to build, train, and deploy custom ML models with full control over the pipeline?
Show Answer
B. Amazon SageMaker. SageMaker provides a fully managed environment for the entire ML lifecycle: data preparation, training, tuning, deployment, and monitoring.
Q12
What is "feature engineering" in machine learning?
Show Answer
B. Creating, transforming, or selecting input variables to improve model performance. Feature engineering transforms raw data into meaningful inputs like extracting day-of-week from timestamps or normalizing scales.
Q13
A company wants to predict house prices based on square footage, location, and age. Which ML task type is this?
Show Answer
B. Regression. Predicting a continuous numerical value (price) from input features is a regression task.
Q14
What is "model drift" and why does it matter?
Show Answer
B. When the statistical properties of input data change, degrading model performance. Real-world data evolves. Continuous monitoring and retraining are essential.
Q15
Which of the following is an example of reinforcement learning?
Show Answer
C. Training a robot to navigate a maze through trial-and-error rewards. Reinforcement learning uses an agent that learns optimal actions by interacting with an environment and receiving reward/penalty signals.
Q16
What does the "precision" metric measure in classification?
Show Answer
B. The proportion of predicted positives that are actually positive. Precision = TP/(TP+FP). High precision means few false alarms.
Q17
Which AWS service uses ML to provide personalized product recommendations?
Show Answer
B. Amazon Personalize. Personalize uses the same technology as Amazon.com to generate real-time recommendations without requiring ML expertise.
Domain 2: Fundamentals of Generative AI (24%) — 20 Questions
Q18
What is the primary architecture used by modern large language models?
Show Answer
C. Transformer. The Transformer architecture uses self-attention mechanisms to process entire sequences in parallel, enabling efficient training on massive datasets.
Q19
What is a "foundation model"?
Show Answer
B. A large model trained on broad data that can be adapted to many downstream tasks. Foundation models learn general representations from massive datasets and can be fine-tuned or prompted for specific applications.
Q20
What is "tokenization" in the context of LLMs?
Show Answer
B. Breaking text into smaller units (tokens) that the model processes. Most LLMs use subword tokenization (like BPE) to balance vocabulary size with representation efficiency.
Q21
What is an "embedding" in AI?
Show Answer
B. A dense numerical vector that represents the meaning of text, images, or other data. Embeddings capture semantic relationships: similar concepts have vectors close together in the embedding space.
Q22
What is "hallucination" in the context of generative AI?
Show Answer
A. When a model generates content that is factually incorrect or fabricated but presented as true. Mitigation strategies include RAG, grounding, temperature reduction, and human review.
Q23
Which training approach teaches a foundation model on vast unlabeled text data to predict the next token?
Show Answer
C. Self-supervised pre-training. The model learns language patterns by predicting masked or next tokens in massive text corpora. No human labels needed.
Q24
What is the "attention mechanism" in Transformers?
Show Answer
B. A mechanism that lets the model weigh the importance of different parts of the input when generating each output token. Self-attention computes relevance scores between all positions in a sequence.
Q25
What is "prompt engineering"?
Show Answer
B. Crafting input text to guide a model's behavior and improve output quality. Techniques include zero-shot, few-shot examples, chain-of-thought reasoning, and system prompts.
Q26
What distinguishes "few-shot" prompting from "zero-shot" prompting?
Show Answer
B. Few-shot includes examples in the prompt; zero-shot provides only the task instruction. Few-shot demonstrates desired format/behavior through examples without parameter updates.
Q27
What is "fine-tuning" a foundation model?
Show Answer
B. Further training a pre-trained model on a smaller, task-specific dataset to adapt its behavior. Fine-tuning updates model weights to specialize in a domain or task with less data and compute than training from scratch.
Q28
What is a "diffusion model" in generative AI?
Show Answer
A. A model that generates images by iteratively removing noise from a random input. Diffusion models learn to reverse a noise-adding process, gradually transforming random noise into coherent images guided by text prompts.
Q29
What is RLHF (Reinforcement Learning from Human Feedback)?
Show Answer
B. A technique that uses human preference rankings to align model outputs with human values. RLHF trains a reward model from human comparisons, then uses RL to optimize the LLM against that reward model.
Q30
What is a "multimodal" foundation model?
Show Answer
B. A model that can process and generate multiple types of data (text, images, audio, video). Multimodal models can understand images alongside text, enabling visual Q&A, document analysis, and image captioning.
Q31
What is the "context window" of an LLM?
Show Answer
B. The maximum number of tokens (input + output) the model can process in a single request. Context windows range from 4K to 200K+ tokens. Larger windows allow processing longer documents but increase compute costs.
Q32
What is "chain-of-thought" prompting?
Show Answer
B. Instructing the model to reason step-by-step before giving a final answer. Chain-of-thought significantly improves performance on math, logic, and multi-step reasoning tasks.
Q33
What is "model distillation"?
Show Answer
B. Training a smaller "student" model to replicate the behavior of a larger "teacher" model. Distillation transfers knowledge into smaller, faster, cheaper models while retaining much of the original performance.
Q34
Which of the following is NOT a typical use case for generative AI?
Show Answer
C. Sorting a database table by column values. Sorting is a deterministic operation handled by traditional algorithms. Generative AI excels at creative, analytical, and language tasks.
Q35
What is "transfer learning"?
Show Answer
B. Applying knowledge learned from one task to improve performance on a different but related task. Foundation models are the ultimate transfer learning: pre-trained on general text, then adapted to specific domains.
Q36
What is a "latent space" in generative models?
Show Answer
B. A compressed, learned representation of data where similar items are close together. In latent space, high-dimensional data is mapped to lower-dimensional vectors that capture semantic meaning.
Q37
What is the key advantage of the Transformer's self-attention over RNNs for sequence processing?
Show Answer
B. Transformers can process all positions in parallel rather than sequentially. RNNs process tokens one at a time. Transformers compute attention across all positions simultaneously, enabling massive parallelization.
Domain 3: Applications of Foundation Models (28%) — 24 Questions
Q38
Which AWS service provides serverless access to multiple foundation models through a unified API?
Show Answer
B. Amazon Bedrock. Bedrock provides serverless API access to foundation models from Anthropic, Meta, Cohere, AI21 Labs, Stability AI, and Amazon.
Q39
What is Retrieval-Augmented Generation (RAG)?
Show Answer
A. A technique that retrieves relevant documents and provides them as context at inference time. RAG grounds model responses in source data, reducing hallucinations and enabling access to current information without retraining.
Q40
A company needs their chatbot to answer questions using proprietary internal documents. Which Amazon Bedrock feature enables this?
Show Answer
B. Knowledge Bases for Amazon Bedrock. This feature implements RAG by automatically chunking, embedding, and indexing your documents, then retrieving relevant passages at query time.
Q41
What does the "temperature" parameter control in LLM inference?
Show Answer
B. The randomness/creativity of the output. Low temperature (0.1-0.3) produces focused, deterministic responses. High temperature (0.7-1.0) increases diversity and creativity.
Q42
What are "Amazon Bedrock Agents"?
Show Answer
B. Autonomous AI systems that can plan, execute multi-step tasks, and invoke APIs/tools on behalf of users. Agents break down requests into steps, determine which tools to call, and synthesize results.
Q43
When should you use fine-tuning instead of RAG?
Show Answer
B. When you want to change the model's style, tone, or behavior patterns rather than add new knowledge. Fine-tuning adapts how the model responds. RAG adds what the model knows.
Q44
What does "top-p" (nucleus sampling) control?
Show Answer
B. The cumulative probability threshold for token selection. Top-p=0.9 means the model considers the minimum set of tokens whose combined probability is 90%.
Q45
A customer service team wants to automatically categorize support tickets by urgency. Which approach is MOST cost-effective?
Show Answer
B. Amazon Comprehend custom classification. For straightforward text classification, Comprehend provides a cost-effective purpose-built solution. Foundation models are overkill for simple classification.
Q46
What is "model selection" in the context of Amazon Bedrock?
Show Answer
B. Evaluating and choosing the right foundation model based on task requirements, quality, speed, and cost. Different models excel at different tasks. Model evaluation helps compare performance.
Q47
What is the purpose of a "system prompt" in LLM interactions?
Show Answer
B. To define the model's persona, constraints, and behavioral instructions. System prompts set the model's role, define output format, restrict topics, and establish guardrails.
Q48
A media company wants to generate product images from text descriptions. Which AWS service is most appropriate?
Show Answer
B. Amazon Bedrock with Stability AI models. Stability AI's image generation models available through Bedrock create images from text prompts and edit existing images.
Q49
What is "prompt injection"?
Show Answer
B. A security attack where malicious instructions override the model's intended behavior. Mitigations include input validation, guardrails, and output filtering.
Q50
What is "semantic search" and how does it differ from keyword search?
Show Answer
B. Semantic search uses embeddings to find results based on meaning similarity. "How do I return a product?" matches "refund policy" in semantic search because the meanings are related.
Q51
Which is a key benefit of Amazon Bedrock over self-hosting open-source models?
Show Answer
B. No infrastructure management, automatic scaling, and pay-per-use pricing. Bedrock eliminates GPU provisioning, scaling, patching, and capacity planning.
Q52
A legal firm wants to summarize 100-page contracts. What should they consider about model selection?
Show Answer
B. The model's context window size must accommodate the full document. A 100-page contract may require 50K-100K+ tokens. Models with small context windows cannot process it in one pass.
Q53
What is the role of a "vector database" in a RAG architecture?
Show Answer
B. Storing document embeddings and enabling fast similarity search. Vector databases index embeddings and efficiently find the most semantically similar documents to a query.
Q54
What is "chunking" in the context of RAG?
Show Answer
B. Splitting large documents into smaller segments for embedding and retrieval. Chunk size affects quality. Too large: diluted relevance. Too small: lost context. Typical sizes: 256-1024 tokens with overlap.
Q55
A company wants their AI assistant to book meetings and query databases. Which Bedrock feature enables this?
Show Answer
C. Agents with action groups. Action groups define available tools (APIs) the agent can call to fulfill user requests like booking meetings or querying databases.
Q56
What is "max tokens" in model inference?
Show Answer
B. The limit on output length the model will generate. The model stops when it hits this limit or produces a stop token. Controls cost and prevents runaway generation.
Q57
When is it appropriate to train a foundation model from scratch?
Show Answer
B. Only when existing models cannot serve the use case. Training from scratch costs millions and takes months. For 99% of use cases, prompting, RAG, or fine-tuning is more practical.
Q58
What is Amazon Titan?
Show Answer
B. Amazon's family of foundation models available through Bedrock. Titan includes text generation, embedding, and image generation models. Titan Embeddings is commonly used for RAG.
Q59
What is "model evaluation" in Amazon Bedrock?
Show Answer
B. Comparing model outputs using automated metrics or human evaluation. Bedrock supports ROUGE, BERTScore, and human evaluation workflows to select the best model for your use case.
Q60
A developer wants to generate code completions. Which factors matter MOST for model selection?
Show Answer
B. Code understanding, reasoning ability, and context window. Large context windows are critical for understanding multi-file codebases and generating contextually correct completions.
Q61
What is "Provisioned Throughput" in Amazon Bedrock?
Show Answer
B. Reserved model capacity that guarantees consistent performance. Provisioned Throughput provides dedicated model units, eliminating throttling for production workloads.
Domain 4: Guidelines for Responsible AI (14%) — 12 Questions
Q62
A facial recognition system has lower accuracy for certain demographic groups. This is an example of:
Show Answer
B. Algorithmic bias. Bias occurs when a system produces systematically unfair outcomes, often due to unrepresentative training data. Requires diverse datasets and fairness metrics.
Q63
Which AWS service helps filter harmful content and enforce responsible AI policies on model inputs and outputs?
Show Answer
B. Amazon Bedrock Guardrails. Guardrails provides content filters (hate, violence, sexual, misconduct), denied topic detection, PII redaction, and word filters.
Q64
What is "explainability" in AI?
Show Answer
B. The ability to understand how a model arrived at a prediction. Techniques include SHAP values, attention visualization, and feature importance ranking.
Q65
What is an "AI Service Card" from AWS?
Show Answer
B. Documentation providing transparency about intended use cases, limitations, and responsible AI considerations. AI Service Cards help customers understand appropriate use cases and known limitations.
Q66
A hiring AI consistently recommends male candidates over equally qualified female candidates. What is the likely root cause?
Show Answer
B. Historical training data reflecting past hiring biases. If training data reflects discrimination, the model learns and perpetuates those patterns. Data curation and bias testing are essential.
Q67
What is the principle of "human-in-the-loop" in responsible AI?
Show Answer
B. Keeping humans involved in oversight for high-stakes outputs. For consequential decisions (medical, legal, financial), human review provides a safety net against errors and bias.
Q68
How does Amazon Bedrock Guardrails handle PII?
Show Answer
B. Detects and can redact or block PII in both inputs and outputs. Guardrails identifies PII types (names, emails, SSNs) and either anonymizes or blocks the response entirely.
Q69
What is "AI transparency"?
Show Answer
B. Openly communicating how AI systems work, their limitations, and when AI is being used. Includes disclosing AI usage, documenting limitations, and providing clear opt-out mechanisms.
Q70
A company deploys AI for loan approvals. Which responsible AI practice is MOST critical?
Show Answer
B. Ensuring fairness, explanations, and bias audits. Financial decisions have legal requirements (ECOA, Fair Housing Act) around non-discrimination and adverse action explanations.
Q71
What is "data poisoning" in AI security?
Show Answer
B. Intentionally introducing malicious examples into training data. An attacker injects biased or backdoored samples to make the model behave incorrectly on specific inputs.
Q72
What is the purpose of "denied topics" in Bedrock Guardrails?
Show Answer
A. Blocking specific topics the model should never discuss. Define custom denied topics with natural language descriptions. The guardrail detects and blocks regardless of how the user phrases the request.
Q73
Which approach BEST mitigates hallucination?
Show Answer
B. Grounding in source documents (RAG) and requiring citations. RAG provides factual context. Citations make hallucination detectable. Low temperature also helps.
Domain 5: Security, Compliance, and Governance (14%) — 12 Questions
Q74
How does Amazon Bedrock handle customer data privacy by default?
Show Answer
B. Customer data is not used to train base models. Your prompts and completions stay within your AWS account, encrypted in transit and at rest.
Q75
A company wants AI traffic to never traverse the public internet. Which AWS feature enables this?
Show Answer
B. VPC endpoints (AWS PrivateLink). PrivateLink creates private connectivity between your VPC and Bedrock without traffic crossing the public internet.
Q76
Which service logs all Amazon Bedrock API calls for audit purposes?
Show Answer
B. AWS CloudTrail. CloudTrail records all Bedrock API calls including who, when, from where, and with which parameters. Essential for compliance.
Q77
How should you control which IAM principals can invoke specific foundation models?
Show Answer
B. IAM policies with bedrock:InvokeModel and model ARNs. Restrict which users/roles can access specific models by specifying ARNs in IAM policy resource elements.
Q78
What encryption does Amazon Bedrock provide for data at rest?
Show Answer
B. AWS-managed keys by default, with customer-managed KMS keys as an option. You can specify your own KMS keys for additional control and compliance requirements.
Q79
A healthcare company must ensure AI outputs comply with HIPAA. Which controls are MOST appropriate?
Show Answer
B. Multiple layered controls. HIPAA requires: BAA with AWS, PII filtering via Guardrails, network isolation via PrivateLink, encryption via KMS, and audit logging via CloudTrail.
Q80
What is "model invocation logging" in Amazon Bedrock?
Show Answer
B. Recording the full input prompts and output completions. Model invocation logging captures what was sent to and received from the model, stored in S3 or CloudWatch for compliance review.
Q81
How can you prevent sensitive data from being included in model prompts?
Show Answer
B. Guardrails PII filters on input plus application-layer redaction. Defense in depth: redact PII before it reaches the model, and configure Guardrails as a safety net to catch anything missed.
Q82
What is the Shared Responsibility Model for AI services like Bedrock?
Show Answer
B. AWS secures infrastructure; customers handle data, access, and compliance. AWS manages the platform security. You manage IAM policies, data classification, guardrails configuration, and regulatory compliance.
Q83
A company wants to ensure only approved foundation models can be used. How should they enforce this?
Show Answer
B. IAM deny policies on unapproved model ARNs. Service Control Policies (SCPs) or IAM policies with explicit deny statements prevent anyone from invoking unauthorized models, regardless of other permissions.
Q84
What role does AWS Config play in AI governance?
Show Answer
B. Evaluating configurations against compliance rules. AWS Config tracks changes to Bedrock resources, verifies compliance with organizational policies, and alerts on non-compliant configurations.
Q85
A financial services company needs to demonstrate to regulators that their AI system makes fair decisions. Which combination provides the strongest evidence?
Show Answer
B. Bias testing, invocation logs, explainability, and audits. Regulators want evidence: documented testing across protected classes, audit trails of decisions, explanations for individual outcomes, and independent validation.
